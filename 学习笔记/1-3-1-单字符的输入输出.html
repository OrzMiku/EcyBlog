1.3.1 单字符的输入输出

|阅读量:

单字符的输入输出

putchar

  • int putchar(int c);
  • 向标准输出写一个字符
  • 返回写了几个字符，EOF(-1)表示写失败

getchar

  • int getchar(void);
  • 从标准输入读取一个字符
  • 返回类型为int，是为了返回EOF(-1)
    • Windows -> Ctrl+Z
    • Unix -> Ctrl+D
